# 1c

A handy devtool for TypeScript / React.

### Installation

```
npm i -g 1c
1c help
```

### Structure

```
path/to/source
+-- html
|   +-- xxx.html
+-- js
|   +-- @types
        +-- yyy.d.ts
|   +-- entries
|       +-- zzz.tsx
+-- package.json (optional)
+-- dist (output)
    +-- xxx.html
    +-- zzz.min.js
```
