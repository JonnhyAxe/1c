# reactc

A handy devtool for React.

### Installation

```
npm i -g reactc
```

### Project Structure

```
path/to/source
+-- html
|   +-- xxxx.html
+-- js
|   +-- @types
|   +-- entries
|       +-- yyyy.tsx
+-- package.json (optional)
+-- dist (output)
    +-- xxxx.html
    +-- yyyy.min.js
```

### Commands

```
reactc watch path/to/source
reactc build path/to/source
```
